# Mac'arroni - Digital Ordering System
This is a custom-built, premium digital dining experience designed to bring the "Mac'arroni" brand to life. I focused on creating a high-impact, dark-themed interface that balances elegant Italian typography with a seamless ordering flow, ensuring the experience is as gourmet as the food itself.

### Key Features
* **Custom UI Design:** A high-end aesthetic using a "Luxury Obsidian" theme with gold accents, perfectly tuned for a gourmet restaurant vibe.
* **Dynamic Cart Logic:** A JavaScript-powered ordering system that handles real-time total calculations, size selections, and a visual "bump" feedback loop.
* **Smart Menu Filtering:** Engineered a fast-filter grid that allows users to switch between Pastas and Pizzas instantly without page reloads.
* **Persistent Auth Flow:** Integrated a secure LocalStorage-based login and registration system to simulate a real-world membership experience.
* **WhatsApp Checkout:** A direct-to-business bridge that formats the user's cart into a professional order message for instant kitchen notification.
* **Device Optimized:** Fully responsive layout using CSS Flexbox and Grid that adjusts perfectly from desktop monitors to mobile phones.

### Why I Built This
Instead of a standard food site, I wanted to demonstrate how a technical solution can solve real-world business problems. I handled everything from the typography hierarchy (Playfair Display for elegance) to the technical execution of the cart state, ensuring the site reflects the premium nature of the "Mac'arroni" brand while providing a friction-less user journey.

### Tech Stack
* **HTML5:** Semantic structure for a solid, SEO-friendly web foundation and accessibility.
* **CSS3:** Custom styling, radial gradients, and advanced positioning for a 3D-layered gourmet effect.
* **JavaScript:** Vanilla JS (ES6+) used for state management, DOM manipulation, and real-time event listeners.
* **Google Fonts:** Integrated 'Playfair Display' and 'Montserrat' to maintain a classic yet modern brand identity.

### How to Use
* Clone the **repository** to your local machine.
* Open the **index.html** file in any modern web browser.
* **No complex installations** or backend servers are required it’s built to be fast and portable.
